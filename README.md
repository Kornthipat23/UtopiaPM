## 🔍 Usage Guide
Once installed, you can use UtopiaPM to manage your packages.

### Basic Commands:
- **List Installed Packages:**
   ```bash
   utopia list
   ```
- **Install a Package:**
   ```bash
   utopia install package-name
   ```
- **Remove a Package:**
   ```bash
   utopia remove package-name
   ```

For more detailed commands, type `utopia help` in your terminal.
